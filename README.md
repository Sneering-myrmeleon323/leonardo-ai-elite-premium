## 🛠️ Common Problems

### The app will not open
- Make sure you extracted the ZIP file first.
- Check that your antivirus did not block the file.
- Right-click the app and choose Run as administrator.
- Download the release again if the file looks damaged.

### The app opens, then closes
- Restart Windows and try again.
- Close other heavy apps.
- Check that your system has enough free memory.

### Images do not generate
- Check your internet connection.
- Confirm that you are signed in.
- Try a shorter prompt.
- Remove special symbols from the prompt.

### The download looks slow
- Try again later.
- Use a wired connection if possible.
- Pause other downloads in your browser.
